# Urban Roots E-Commerce Website

## Branding
- **Website Name:** Urban Roots
- **Main Theme Color:** #006838
- **Secondary Colors:** #55833d & #8cc342
- **Background:** #ffffff (Light Mode)

## Project Structure

### User Pages
- `index.html` - Home Page
- `shop.html` - Shop Page
- `blog.html` - Blog Page
- `profile.html` - User Profile
- `settings.html` - User Settings
- `login.html` - Login Page
- `checkout.html` - Product Checkout
- `payment.html` - Payment Page

### Admin Pages
- `admin.html` - Admin Dashboard
- `admin/manage-products.html` - Manage Products
- `admin/manage-blogs.html` - Manage Blog Posts
- `admin/manage-ads.html` - Manage Advertisements
- `admin/manage-videos.html` - Manage Videos

### Assets
- `css/` - Stylesheets
- `js/` - JavaScript Files
- `images/` - Image Assets
- `data/` - JSON Data Files

## Features
1. Navigation Bar with Logo and Links
2. User Profile with Dropdown Menu
3. Notifications System
4. Favorites/Wishlist
5. Product Listings with Filters
6. Blog Posts Section
7. Advertisement Placement
8. YouTube Video Embedding
9. Newsletter Subscription
10. Admin Dashboard for Content Management
11. Payment Integration Ready
12. Responsive Design

## Installation
1. Clone the repository
2. Open `index.html` in your browser
3. For admin access, navigate to `admin.html`

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON (for data storage)
